# Flujo del negocio actual

En esta sección se resume cómo funcionan actualmente los negocios entrevistados, con énfasis en los pasos manuales y en los puntos donde aparecen errores, retrasos o desorden.

## Restaurante Don Juan

### Flujo actual

1. El cliente llega al restaurante o solicita un pedido.
2. El mesero toma la orden en papel.
3. El mesero se traslada físicamente a cocina para entregar el pedido.
4. El cocinero interpreta la orden y comienza la preparación.
5. El mesero verifica si el plato ya está listo.
6. El pedido se entrega al cliente.
7. El cliente solicita la cuenta.
8. El total se anota manualmente.
9. El cliente paga y la venta se registra en un cuaderno.

### Puntos de fricción detectados

| Etapa | Riesgo observado |
| --- | --- |
| Toma del pedido | Errores por escritura manual o pérdida del papel |
| Comunicación con cocina | Retrasos por traslado físico del mesero |
| Seguimiento del pedido | Falta de visibilidad del estado en tiempo real |
| Cobro | Riesgo de inconsistencias en la cuenta |
| Registro de ventas | Información dispersa y difícil de consolidar |

## Clínica dental

### Flujo actual

1. El paciente agenda una cita por teléfono.
2. La recepcionista registra la cita en papel.
3. Llega el día de atención sin recordatorio automatizado.
4. Si el paciente llega, se busca el historial clínico en archivos físicos.
5. El odontólogo atiende la consulta.
6. El pago se anota manualmente.
7. Si el paciente no llega, se pierde la cita y luego se intenta reagendar.

### Puntos de fricción detectados

| Etapa | Riesgo observado |
| --- | --- |
| Agenda | Doble asignación o pérdida de citas |
| Recordatorio | Ausencia de avisos previos al paciente |
| Historial clínico | Dificultad para localizar expedientes físicos |
| Cobro | Registros manuales poco confiables |
| Reagendación | Tiempo perdido por citas incumplidas |

## Hallazgo principal

:::warning Conclusión del análisis
Aunque ambos negocios mostraron oportunidades de mejora, el restaurante presentó un flujo más visible y con más pasos manuales críticos para el proyecto, por lo que se convirtió en el foco principal de la propuesta.
:::
